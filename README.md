# Vuks-cogs
This repo contains cogs for
[Red-Discordbot](https://github.com/Cog-Creators/Red-DiscordBot) V3.

# Installation
Use Downloader:

`[p]repo add vuks https://github.com/Vuks69/Vuks-cogs`

`[p]cog install vuks <cog>`
